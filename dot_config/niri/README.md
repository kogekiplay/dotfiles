# Niri

滚动式 Wayland 合成器配置。

## 文件说明

- `config.kdl` - 主配置文件
- `cfg/` - 模块化配置片段
  - `animation.kdl` - 动画设置
  - `autostart.kdl` - 自启动应用
  - `display.kdl` - 显示设置
  - `input.kdl` - 输入设备设置
  - `keybinds.kdl` - 快捷键绑定
  - `layout.kdl` - 窗口布局
  - `misc.kdl` - 其他设置
  - `rules.kdl` - 窗口规则

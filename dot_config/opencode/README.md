# OpenCode

AI 代码助手配置。

## 文件说明

- `opencode.json` - 主配置文件（已排除提交，包含 API key）
- `dcp.jsonc` - DCP 插件配置
- `opencode-mem.jsonc` - 记忆插件配置
- `tui.json` - TUI 界面配置
- `package.json` - Node.js 包配置
- `commands/` - 自定义命令
- `themes/` - 主题
- `plugins/` - 插件
- `node_modules/` - Node.js 依赖（已排除提交）

# Kando

环形快捷菜单配置。

## 文件说明

- `config.json` - 主配置文件
- `menus.json` - 菜单配置
- `achievements.json` - 成就数据
- `electron-flags.conf` - Electron 启动参数
- `ipc-info.json` - IPC 通信信息
- `icon-themes/` - 图标主题
- `menu-themes/` - 菜单主题
- `sound-themes/` - 声音主题
- `session/` - 会话数据（已排除提交）

# Fish Shell

Fish Shell 配置。

## 文件说明

- `config.fish` - 主配置文件
- `conf.d/` - 自动加载的配置片段
  - `env.fish` - 环境变量加载（从 ~/.env）
  - `rustup.fish` - Rust 工具链环境
  - `uv.env.fish` - UV Python 包管理器环境
- `completions/` - 自动补全脚本
- `functions/` - 自定义函数
- `fish_variables` - Fish 内部变量（不建议手动编辑）

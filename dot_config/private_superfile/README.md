# Superfile

终端文件管理器配置。

## 文件说明

- `config.toml` - 主配置文件
- `hotkeys.toml` - 快捷键配置
- `theme/` - 主题文件
  - `catppuccin-*.toml` - Catppuccin 主题系列
  - `dracula.toml` - Dracula 主题
  - `gruvbox*.toml` - Gruvbox 主题
  - `nord.toml` - Nord 主题
  - `tokyonight.toml` - Tokyo Night 主题
  - 等等...

# Noctalia

Noctalia 桌面栏配置。

## 文件说明

- `settings.json` - 主配置（主题、布局）
- `colors.json` - 颜色配置
- `plugins.json` - 插件配置
- `plugins/` - 插件目录
  - `file-search/` - 文件搜索
  - `github-feed/` - GitHub 动态（已排除 token）
  - `kaomoji-provider/` - 颜文字
  - `kde-connect/` - KDE Connect 集成
  - `notes-scratchpad/` - 便签
  - `screen-recorder/` - 录屏
  - `screenshot/` - 截图
  - `translator/` - 翻译

# Noctalia Morandi Theme

Noctalia 5 uses TOML config. This directory keeps the active Noctalia config plus a small Morandi color pipeline derived from the current wallpaper.

## Files

- `settings.toml`: Noctalia 5 config for this Arch+niri setup.
- `apply-morandi.sh`: Hook called by Noctalia when wallpaper colors change.
- `morandi-gen.py`: Generates low-saturation Morandi colors for desktop tools.
- `colors.json`: Generated by the hook and safe to overwrite.

## Managed Targets

The generator updates:

- niri focus ring and tab indicator colors
- fcitx5 theme assets and `classicui.conf`
- starship palette
- fastfetch theme colors
- alacritty colors
- cava Morandi theme
- Qt/KDE color scheme

Proxy clients, bootloader artwork, and private app state are intentionally not touched.
